<h1>💻 Projeto Pokédex</h1>
<p>O projeto consiste em uma plataforma web que tem como funcionalidade listar todos os Pokémon exibir seus atributos. Todos os dados sobre os pokémon foram obtidos através da API REST <a href="https://pokeapi.co/" target="_blank">PokéApi</a>.
Grande parte visual do projeto foi inspirado em um <a href="https://www.behance.net/gallery/95727849/Pokdex-App" target="blank">layout</a> de um app sobre Pokémon.
<p>
<p>🌐 Link do Projeto - https://pokedex-carloszaparoli.vercel.app/</p>

<h3>Principais Funcionalidades</h3>
<ul>
  <li>
    <strong>Listagem de Pokémon:</strong>
    Lista os 15 primeiros pokémon retornados da API.
  </li>
  <li>
    <strong>Carregar mais Pokémon:</strong>
    O método consulta a API e lista os próximos 15 pokémon.
  </li>
  <li>
    <strong>Filtro da lista:</strong>
    A lista pode ser filtrada tanto pelo nome do pokémon quanto pelo tipo do pokémon, ou a combinaçao de ambos.
  </li>
  <li>
    <strong>Alteração de tema:</strong>
    Através de um botão na parte superior direita, é possível alterar o tema da aplicação para Dark (tema escuro) ou para Light (tema claro). O tema selecionado fica salvo no localStorage.
  </li>
</ul>

<h1>📥 Instalação e Execução</h1>
<p>Faça o clone desse repositório e acesse o diretório</p>
<pre>
$ git clone https://github.com/carloszaparoli/pokedex.git && cd pokedex
</pre>
<span>Instalando dependências:</span>
<pre>
$ npm install
</pre>
<span>Executando a aplicaçao:</span>
<pre>
$ npm run dev
</pre>
